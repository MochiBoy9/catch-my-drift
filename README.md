# catch my drif
a small web-based application that (hopefully ) is your controller's ally

# controls
the website will NOT work unless you have a controller connected to your computer, phone, etc. the controller can be connected wired or bluetooth, does not matter. 

once you got a controller plugged in, press a button or two and the app should start running.

# features
- button checking
- stick checking/location tracking
- trigger sensitivity checks
